
# Spreadsheet


## Features

- Write in a cell
- Sum cells with `=SUM(cell1,cell2, ... cellN)` e.g.: `=SUM(B2,C3)`
`=MULTIPLY(cell1,cell2, ... cellN)` e.g.: `=MULTIPLY(B2,C3)`


## Tech Stack

**Client:** Sveltekit + TailwindCSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/Gautierpicon/Spreadsheet
```

Go to the project directory

```bash
  cd Spreadsheet
```

Install dependencies

```bash
  bun install
```

Start the server

```bash
  bun run dev
```
